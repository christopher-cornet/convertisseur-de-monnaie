# Convertisseur De Monnaie
Ce programme permet de convertir de l'argent d'une devise à une autre.
<br><br>
<b>Exemple :</b>
```
Pour convertir Euro vers Dollar = Euro * 1.08
Car 1€ = 1.08$
```
